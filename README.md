# Video Prep - Delivery Service


### Technologies
* Written in Express.js,
* Tested with Mocha and Chai
* Continuous integration: Circle CI
* Containerized with Docker 
* Deployed to AWS EC2 instance
* Logs/Analytics: Elassticsearch, Kibana

### For development
```
npm i
npm run dev
```
